# Hepsiburada QA Automation Framework

This project is a professional-grade UI automation framework for testing **Hepsiburada.com**'s filtering and search functionalities.

## 🛠 Tech Stack
* **Language:** Java 11/17
* **Framework:** TestNG
* **Automation Tool:** Selenium WebDriver
* **Design Pattern:** Page Object Model (POM)
* **Build Tool:** Maven

## 🚀 Key Features
* **Page Object Model:** Clean and maintainable code structure.
* **Synchronization:** Utilizes Explicit Waits (WebDriverWait) instead of hard-coded sleeps.
* **Cross-browser Support:** Easy to configure for Chrome, Firefox, or Edge.
* **Reporting:** Integrated with TestNG for execution results.

## 🏁 Getting Started
1. Clone the repository: `git clone https://github.com/Fatmanur-dinc/Hepsiburada-QA.git`
2. Install dependencies: `mvn clean install`
3. Run tests: `mvn test`
